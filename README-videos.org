** 024 Web Search Integration - @search() inline syntax #emacs #ollama

Demonstrating the Emacs package ollama-buddy's web search integration.

Select cl:minimax-m2.1:cloud

- Open chat and type: =What is the next fixture for Manchester United=
  Check that the LLM knows nothing about emacs 31
- Open chat and type: =What is the @search(next fixture for Manchester United)=
- Show the ♁N indicator appearing in the header line
- Use =C-c C-w= to show the attached web search results
- Ask a follow-up question using the attached context
  "and what about liverpools" - to prove that we only have Manchester United context
- type: =and what about @search(liverpools next fixture?)=
- Use =C-c C-w= to show the attached web search results
- Use =C-c 0= to clear all attachments

** 036 Git Repository Operations with Tool Calling #emacs #ollama

Demonstrating git repository operations through ollama-buddy tool calling.

- Open chat with a model that supports tool calling: glm-4.7:cloud
- Turn on tooling
- "Can you check the git status of the ollama-buddy repository?"
- Watch execute_shell tool calling git status
- "Now create a dummy test file in the repository for stash demonstration"
- Watch write_file tool create a test file
- "Stage the file and create a stash with a descriptive message"
- Watch execute_shell run git add and git stash push
- "List all stashes in the repository"
- Watch execute_shell show the stash list
- "Now delete the most recent stash"
- Watch execute_shell run git stash drop
- "Verify the stash was deleted by listing stashes again"
- Watch execute_shell confirm the deletion

This demonstrates how ollama-buddy can interact with git repositories using tool calling to perform version control operations like checking status, creating stashes, listing stashes, and deleting them - all through natural language commands.
